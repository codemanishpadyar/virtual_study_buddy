# StudyBuddy
## Project Description
StudyBuddy is a web-based application designed to provide users with an AI-powered study companion. The application offers a user-friendly interface for users to interact with the study buddy, manage their study tasks, and schedule.

## Features
* AI-powered study buddy with OpenAI GPT model integration
* User-friendly interface for interacting with the study buddy
* Study planner interface for managing study tasks and schedule
* Login system to ensure secure access to the application
* Demo mode for testing the application without an API key

## Tech Stack
* Frontend: HTML, CSS, PHP
* Backend: PHP, OpenAI GPT model
* Database: MySQL (not included in this repository)
* External dependencies: Google Fonts, Font Awesome

## Installation
1. Clone the repository to your local machine.
2. Create a database and update the database connection settings in the `studybuddy_config.php` file.
3. Set your OpenAI API key in the `studybuddy_config.php` file.
4. Upload the files to your web server.

## Usage
1. Access the application through the `index.php` file.
2. Log in to the application using your credentials.
3. Interact with the study buddy through the `studybuddy.php` file.
4. Manage your study tasks and schedule through the `study_planner.php` file.

Note: This repository does not include the database schema or the OpenAI API key. You will need to set up your own database and obtain an API key to use the application.
you can get your api key from: 
https://platform.openai.com/api-keys
